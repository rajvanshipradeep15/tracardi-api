# List of available form field types 

The following field types can be used in plugin's forms:

* readOnlyTags
* eventTypes
* eventType
* resource
* dotPath
* forceDotPath
* keyValueList
* copyTraitsInput
* listOfDotPaths
* text
* json
* sql         
* textarea      
* select
* bool          
* contentInput

The list of form fields is getting bigger with every Tracardi release.
